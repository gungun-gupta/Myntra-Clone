# 🛍️ Myntra Clone

A responsive front-end clone of the Myntra website built with **HTML**, **CSS**, and **JavaScript**. This project replicates the core UI/UX of the popular e-commerce platform with a focus on clean design, responsiveness, and user interactivity.

---

## 🔥 Key Features

- Responsive header with logo, nav menu, and search bar  
- Mega dropdown for easy navigation  
- Wishlist, Profile, and Bag sections  
- Hover effects and smooth transitions  
- Dynamic product listings using JavaScript/JSON  
- Placeholder setup for login and cart features  
- Clean and modern footer with social links  
- Mobile-friendly layout with media queries  

---

## 🚀 Tech Stack

- **HTML5**  
- **CSS3** (Flexbox, Media Queries)  
- **JavaScript** (DOM Manipulation, Event Handling)
